üîë Environment Variables

To run this project, you need to add the following environment variables to your .env file. The base project already includes a .env file, but please ensure these variables are added or modified as needed:

VITE_HOST_ADDR = 'localhost'
VITE_PORT_API = 4000
VITE_PORT_APP = 5173
VITE_API_PATH = 'api'

üó≥Ô∏è Prerequisites

Before you begin, ensure you have Node.js installed using the following commands or using a executable from here.

Windows (PowerShell)

winget install Schniz.fnm
fnm install 23
node -v
npm -v

Linux / MacOs (Bash)

curl -o- https://fnm.vercel.app/install | bash
fnm install 23
node -v
npm -v

üîß Run Locally (Development Mode)

  1. Clone the project
  git clone https://github.com/andrei-v-stan/watr.git
  1. Navigate to project directory
  cd watr
  1. Install dependencies
  npm install

  1. Start the project
    (You'll need to run two commands in separate terminals / one concurrent command for development mode)

4.a. Terminal 1 (/server | server.js):

  npm run dev-server

4.b. Terminal 2 (/src | vite):

  npm run dev-client

‚öôÔ∏è Deployment Mode

  1. Clone the project
  git clone -b gcloud https://github.com/andrei-v-stan/watr.git
  1. Navigate to project directory
  cd watr
  1. Install dependencies (packages.json)
  npm install
  1. Build the project (vite)
  npm run build
  1. Start the project (index.js)
  npm run start
